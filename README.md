# Java-Based-Atm-Simulator-System
My final year project is an ATM Simulator System developed using Java, Java Swing, JDBC, MySQL, and JWT authentication. The project is designed to simulate the working of a real ATM machine and provide users with a simple and secure banking experience.

In this system, users can create an account, log in securely, check their account balance, deposit money, withdraw cash, transfer funds, change their PIN, and view transaction history. The graphical interface is created using Java Swing, which makes the application easy to use and interactive. MySQL database is used to store user details and transaction records, while JDBC is used to connect the application with the database. JWT authentication is implemented to improve security and protect user accounts from unauthorized access.

This project helped me improve my knowledge of Java programming, database connectivity, GUI design, and authentication systems. It also gave me practical experience in developing a real-world banking application with secure transaction management.
